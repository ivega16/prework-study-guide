Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body element contains a header, main, and footer section.
  • Sections can be used to separate different parts of a webpage.

CSS

  • a CSS rule is defined by a selector and declaration block betwen brackets, containing properties and values.
  • a link element creates the connection to the external file.
  • the rel attribute specifies the relationship between the current and linked doc resource.
  • the href attribute specifies the location (URL) of the external resource.
  • a class selector (ex: .card) can be used as a HTML class attribute (ex: card)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on.
  • git checkout main: switches us back to the main branch
  • git pull origin main: to pull down the latest main branch
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: to commit all files in the working branch
  • git commit -m "message": to commit changes to repo and we add -m flag to associate message with commmit.
